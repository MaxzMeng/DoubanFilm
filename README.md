## DoubanFilm
一款豆瓣电影Android客户端
后端依赖于[DoubanSpider](https://github.com/MaxzMeng/DoubanSpider)
## 运行
### 直接运行
我已经在服务器上部署后端，直接运行即可看到效果
### 自己运行
自行搭建后端环境并将NetWork类中的baseUrl修改为你自己的即可运行
