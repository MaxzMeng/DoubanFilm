## DoubanFilm
一款豆瓣电影Android客户端
后端依赖于[DoubanSpider](https://github.com/MaxzMeng/DoubanSpider)
##运行
目前我并没有在任何服务器上搭建该项目的后端，如果想要运行请自行搭建后端环境并将NetWork类中的baseUrl修改为你自己的即可运行
